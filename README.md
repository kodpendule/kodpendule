# Kod Pendule

The webshop project and Python virtual environment live in **`kodpendule/`**.

```powershell
cd kodpendule
.\.venv\Scripts\Activate.ps1
python manage.py runserver
```

Do not create `venv` in this parent folder — use `kodpendule\.venv` only.

See [kodpendule/README.md](kodpendule/README.md).
